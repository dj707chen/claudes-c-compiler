Add missing standard C builtin macros and typedef-like definitions:

1. Add NULL macro (defined as ((void*)0)) - critical, almost every program uses it
2. Add size_t, ptrdiff_t, wchar_t as builtin typedef-like macros
3. Add va_list/va_start/va_end/va_arg macros for <stdarg.h>
4. Add additional missing stdlib/stdio macros (FILENAME_MAX, FOPEN_MAX, etc.)
5. Add __STDC__, __STDC_VERSION__, __STDC_HOSTED__ macros
6. Add __SIZEOF_* macros for type sizes

These are causing widespread test failures because tests include standard headers
and expect these symbols to be defined.
