Refactor i686 linker: break up monolithic 3200-line elf.rs

The i686 linker has a single 2400-line link_builtin() function that handles
everything from argument parsing through ELF emission. This task will:

1. Break link_builtin() into focused functions matching x86/ARM patterns:
   - parse_args, load_files, resolve_archives, merge_sections,
   - resolve_symbols, build_plt_got, layout_segments,
   - apply_relocations, emit_elf
2. Remove the duplicate DynStrTab (use linker_common's version)
3. Remove duplicate elf_hash wrappers (use linker_common directly)
4. Add proper documentation

This is a refactor-only change with no behavioral modifications.
