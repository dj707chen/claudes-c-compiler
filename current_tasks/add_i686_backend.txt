Task: Add i686 (32-bit x86) backend
Status: in_progress
Started: 2026-01-28

Goal: Add initial i686/x86-32 backend support to the compiler.

This is a large task that will be done incrementally. First step:
- Add Target::I686 variant to the Target enum
- Create i686 backend directory structure (codegen, assembler, linker modules)
- Implement basic i686 codegen by adapting x86-64 backend:
  - 32-bit registers (EAX-EDI, no R8-R15)
  - cdecl calling convention (all args on stack)
  - 32-bit pointers and addressing
  - No REX prefix handling
  - Stack-based parameter passing
- Add ccc-i686 binary to Cargo.toml
- Add i686 detection in main.rs
- Set correct predefined macros for i686 target
